# simple-meal-history-table

A very very simple website with only one page, which showcases a table that represents the information retrieved from the Personal_MealHistory table in the database. The info is retrieved via API call request to the database.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/simple-meal-history-table.git
cd simple-meal-history-table
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
